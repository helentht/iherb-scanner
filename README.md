# iherb-scanner

This is a web application which allows users to scan the barcode of a product and check the current price on iherb.com.

## How to run:

- ### Backend:

```shell
cd iherb-scanner/backend
```

```shell
python3 -m venv venv
```

```shell
source venv/bin/activate
```

```shell
flask run
```

- ### Frontend:

```shell
cd iherb-scanner/
```

```shell
npm install
```

```shell
npm start
```
